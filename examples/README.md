# Examples

These are illustrative RGDS decision records intended to show how the Decision Log schema is used in practice.

- `rgds-dec-0001.json` — Conditional-go decision with explicit evidence, conditions, governance approvals, and bounded AI assistance disclosure.
- `rgds-dec-0002-no-go.json` — No-go decision demonstrating defensible stopping criteria, evidence-based risk posture, and remediation actions before re-review.
